| Term      | Explain                           |
| --------- | --------------------------------- |
| Scene     | 영화의 장면 장면을 생각하면 된다. |
| Inspector | 오브젝트 세부 설정                |
|           |                                   |

#### ○ Object Inspector

| Term          | Explain            |
| ------------- | ------------------ |
| Mesh Renderer | 오브젝트 질감 설정 |
|               |                    |
